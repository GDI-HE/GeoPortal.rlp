Generic JsonResponse wrapper for Geoportal

Use for AJAX responses. There are three default values for the response: ‘html’, ‘response’ and ‘url’. ‘Html’ contains prerendered html content, that will be pasted by Javascript into an html element.

IMPORTANT: Always(!) use this object instead of a direct JsonResponse() object.
